## Getting Started

### CMake

- Extract the zip of this project
- Type the following commands:

```sh
cmake -S . -B build
cmake --build build
```
### Run

``` sh
./build/chess-fow/chess-fow
```

## Description

Chess with (upcoming) fog of war mode.
